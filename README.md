# GabsMedical

## Description
Usando o Microprocessador STM32F030F4P6 ARM Cortex M0 iremos implementar um dispositivo monitor de frequência cardíaca.


## Componentes
* Cortex M0 - STM32F030F4P6
* NodeMCU - ESP32  
* Sensor de Frequência Cardíaca
